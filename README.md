# MDD Booster

MDD 를 지원하는 코드 생성기 입니다.

**설치**

> `dotnet tool install mdd-booster --glob`

**업데이트**

> `dotnet tool update mdd-booster --global --no-cache`
<br> --no-cache 가 붙여있어야 최신을 받을 수 있음

**실행**

> `mdd <directory-path>`
<br> // `directory-path` 매개변수로 실행

> `mdd` // 현재경로

**Settings**

`settings.json` 파일이 필요합니다.

```
{
}
```
